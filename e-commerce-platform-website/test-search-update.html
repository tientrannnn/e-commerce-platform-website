Test Search Update Functionality

✅ Đã sửa xong!

Vấn đề: Khi search lại ở trang search, không cập nhật kết quả mà phải reload

🔧 Những gì đã sửa:

  • SearchView.vue: Thêm watch để theo dõi thay đổi route.query.query
  • NavbarView.vue: Sử dụng router.replace() thay vì router.push()
  • Reset state: Reset searchResults và displayedCount khi search mới

🧪 Cách test:

  1. Mở http://localhost:5173
  2. Search một từ khóa (ví dụ: "Chuột")
  3. Ở trang search, thay đổi query trong URL thành từ khóa khác (ví dụ: "sd")
  4. Nhấn Enter hoặc click search
  5. Kiểm tra xem kết quả có cập nhật ngay lập tức không
  6. Không cần reload trang nữa!

📝 Code đã thêm:

// SearchView.vue
import { ref, onMounted, computed, watch } from 'vue'

// Watch for route query changes to reload search
watch(() => route.query.query, (newQuery, oldQuery) => {
  console.log('Route query changed:', { newQuery, oldQuery })
  if (newQuery && newQuery !== oldQuery) {
    console.log('Query changed, performing new search:', newQuery)
    searchQuery.value = newQuery
    performSearch(newQuery)
  }
}, { immediate: true })

// NavbarView.vue
const handleSearchEnter = () => {
  if (searchQuery.value.trim().length >= 2) {
    router.replace(`/search?query=${encodeURIComponent(searchQuery.value.trim())}`)
  }
}

🔄 Flow hoạt động:

  1. User search từ navbar → router.replace() → URL thay đổi
  2. SearchView.vue watch phát hiện thay đổi query
  3. Gọi performSearch() với query mới
  4. Reset state và fetch data mới
  5. Hiển thị kết quả mới ngay lập tức
